## 开始使用GitHub
### 初始化仓库
进入需要存放文件的本地目录下，使用命令
> git init
初始化本地储存库

### 拷贝Github库到本地
使用命令
> git clone <Github库的链接>
- 在网页上点击 Clone or Download 会显示

### 添加文件
使用命令 
>git add <文件名>
### 设置用户名
>git config --global user.email "you@example.com"
>git config --global user.name "Your Name"

### 提交更改
使用命令
>git commit -m "一些描述"

### 设置上传地址
>git remote add <Github库的链接>

### 上传
>git push
